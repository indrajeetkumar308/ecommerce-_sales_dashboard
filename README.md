# 🛒 Rajan Ecommerce Sales Dashboard
An interactive Power BI sales analytics dashboard built to track and visualize key ecommerce performance metrics — including revenue, profit, quantity sold, and customer behavior — across categories, states, and time periods.

# 📊 Dashboard Overview
This single-page Power BI report consolidates ecommerce order and transaction data into a clean, dark-themed dashboard that allows stakeholders to quickly assess business health and drill into specific segments.
KPI Cards

# Total Amount — Overall revenue generated
Total Profit — Net profit across all orders
Total Quantity — Units sold
Average Order Value (AOV) — Average spend per order
